# Directory structure

 * `images`: diagrams in PNG format exported from modelio-project.zip
 * `modelio-project.zip`: Bundled [modelio](https://www.modelio.org) project with UML diagrams.

To update diagrams do next steps:
 * `unzip modelio-project.zip` . `modelio-yoroi` folder will appear.
 * In Modelio: "File"->"Import a project" and select `modelio-yoroi` folder.
